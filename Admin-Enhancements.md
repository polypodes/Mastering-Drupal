Drupal Admin Enhancements
=========================

Initial modules
---------------

In /admin/modules, we:

* install & enable [admin_menu](https://drupal.org/project/admin_menu)
* set as inactive:  Core > "Toolbar" module
* set as active: Administration > "Administration menu Toolbar style" module
